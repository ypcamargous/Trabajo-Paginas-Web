INFORMACIÓN DEL CURSO

Levantamiento y Análisis De Requerimientos De Software

La ingeniería de requisitos del software es un proceso de descubrimiento, refinamiento, modelado y especificación. Se afinan en detalle los requisitos del sistema y el papel asignado al software.

El análisis de requisitos es una tarea de ingeniería del software es parte de vital de la definición del software a nivel sistema y el diseño de software. El análisis de requerimientos permite al ingeniero de sistemas especificar las características operacionales del software (función, datos y rendimientos), indica la interfaz del software con otros elementos del sistema y establece las restricciones que debe cumplir el software.

Al finalizar el curso, serás capaz de:

OBJETIVOS

1. Adquirir conocimientos conceptuales , metodológicos necesarios para realizar el levantamiento de la información, Identificación y evaluación de la necesidad.

2. Identificar las técnicas y herramientas para el análisis de requisitos con la aplicación de estándares y principios de software.

3. Aplicar técnicas de recolección de información para el levantamientos de requerimientos funcionales y no funcionales, así como la especificación de los mismos.
